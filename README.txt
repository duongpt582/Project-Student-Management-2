# Project-Student-Management
This project uses:
 - IDE: Apache Netbeans
 - JDK: 16
 - Special library:
    + KControls-2.0.jar : Use as a color gradient interface
    + mysql-connector-java-8.0.28.jar: database connection
    + jcalendar-1.4.jar: use to select date of birth
    + AbsoluteLayout.jar: make design easier
